# Railway Track Simulation
A web-based simulation of a 5km railway track with sensors, UAV monitoring, and fault detection.

## Features
- Interactive map with track and sensors.
- Real-time UAV telemetry and fault alerts.
- Heatmap for sensor health.
- Realistic noise, latency, and packet loss.

## How to Run Locally
1. Clone the repo: `git clone https://github.com/yourusername/RailwayTrackSimulation.git`
2. Open `index.html` in a browser or use VS Code's Live Server.

## Technologies
- HTML, CSS, JavaScript
- Leaflet.js for maps
- Chart.js for charts
