# Railway Track Simulation
A web-based simulation of a 5km railway track with sensors, UAV monitoring, and fault detection.

## Features
- Interactive map with track and sensors.
- Real-time UAV telemetry and fault alerts.
- Heatmap for sensor health.

## How to Run
1. Open in VS Code.
2. Right-click `index.html` and select "Open with Live Server".
3. View in browser.

## Technologies
- HTML, CSS, JavaScript
- Leaflet.js for maps
- Chart.js for charts
